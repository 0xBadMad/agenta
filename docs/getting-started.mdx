To build an app that can be deployed and tested with Agenta you need to follow a couple of things:

## API-keys
You need to save your keys in the .env file
Then you need to load these in your python script using the dotenv library
```python
from dotenv import load_dotenv
load_dotenv()
```

## The app structure
### Example App Structure

Here's an example of how your app folder might look:

```bash
my_agenda_app/
│
├── .env
├── app.py
└── requirements.txt
```

### Example app.py
```python
from agenta import post, get

class ChatInput(BaseModel):
    messages: List[str]

class ChatOutput(BaseModel):
    response: str

@post
def chat(input_data: ChatInput) -> ChatOutput:
    # Your chat function implementation here
    response = "Hello, World!"
    return ChatOutput(response=response)

class IngestInput(BaseModel):
    data: List[str]

class IngestOutput(BaseModel):
    status: str

@post
def ingest(input_data: IngestInput) -> IngestOutput:
    # Your ingest function implementation here (if needed)
    status = "Data ingestion successful."
    return IngestOutput(status=status)

```
